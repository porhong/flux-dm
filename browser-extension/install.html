Install FluxDM Browser Integration

The FluxDM installer has already registered the secure native messaging host. Complete these browser steps once for Chrome and/or Edge.

Chrome

  1. Open chrome://extensions.
  2. Enable Developer mode.
  3. Select Load unpacked.
  4. Select the folder containing this setup page and manifest.json.
  5. Open FluxDM Browser Integration → Details → Extension options, then select Test connection.

Microsoft Edge

  1. Open edge://extensions.
  2. Enable Developer mode.
  3. Select Load unpacked.
  4. Select the folder containing this setup page and manifest.json.
  5. Open the extension options and select Test connection.

Keep FluxDM installed at its registered location. Automatic interception can be disabled or excluded by host and file extension. Browser downloads are cancelled only after FluxDM accepts them.
